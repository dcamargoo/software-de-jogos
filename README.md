# Software de Jogos
Projeto de Banco de Dados

Uma empresa está desenvolvendo uma plataforma para venda de jogos online
onde os usuários podem comprar, descobrir, avaliar e adquirir expansões para
jogos. Eles precisam de um banco de dados para gerenciar as informações
relacionadas aos jogos, avaliações dos usuários e expansões disponíveis, a fim de
ter um controle sobre seu sistema e tentar obter maiores lucros no futuro.

Esse projeto inicia com a modelagem do Banco de Dados e se extende até análises com auxílio de gráficos.
